**AI Expense Analyzer**
Smart Expense Categorization | Financial Insights | Forecasting

An end-to-end Data Science + Machine Learning + Data Analytics project that helps users automatically categorize expenses, analyze financial trends, and forecast future spending.
Built using Python, Streamlit, Machine Learning, and Power BI.

**Machine Learning Workflow**
    Step	                          Description
Data Preprocessing	        Cleaned expense & income data
Feature Engineering	        Combined “account + tag” → TF-IDF
Model Training	            Random Forest Classifier
Prediction Output	          Suggested Spending Category
Time-Series Forecasting	    Statsmodels ARIMA model

**Tech Stack**
Category	                   Tools
Frontend App	            Streamlit
ML Modeling	         Scikit-Learn, Statsmodels
Processing	         Python, Pandas, NumPy
Visualization	       Matplotlib, Power BI
Deployment	             Streamlit Cloud
